# Wintune_Uninstall_Search

Sometimes, when I am publishing a new Windows App in Microsoft Intune, I need to know the exactly detection rule MSIexec key or the Uninstall command. These values are in the regedit and my memory is bad... :(

For this reason, I created this simple PowerShell script that request the name of the app in graphical mode and share with you a table with all these values...
